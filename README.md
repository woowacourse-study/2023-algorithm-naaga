# 코코닥 선생님의 알고리즘 스터디
#### 기간 : [레벨 4] 2023.8.21 ~

## 🔍 스터디 목적
#### 알고리즘, 자료구조에 대한 학습한다.
#### 알고리즘 별 문제 풀이를 통해 해당 알고리즘의 이해도를 높인다.

## 💡 규칙
### 1. 알고리즘 주제 선정 및 문제 선정
#### 코코닥 주도로 주마다 1개의 알고리즘 주제 선정
#### 해당 알고리즘 별 5문제 선정
#### 매주 월요일 14시에 1시간 동안 스터디를 진행한다.
#### 지난 주 선정 문제를 모두 풀어온다.

### 2. 문제 제출
